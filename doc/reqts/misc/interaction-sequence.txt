title Robosport Interactions

Main Menu->Team Select: Watch Mode
Main Menu->Team Select: Instant Mode
Main Menu->Team Select: Test-Bench Mode

Team Select->Team Select: Load robot

Team Select->Game View: Start (if in Watch mode)
Team Select->Test View: Start (if in Test-Bench Mode)
Team Select->Results Page: Start (if in Instant mode)

Game View->Game View: Turns execute
Game View->Game View: Step through actions
Test View->Test View: Turns execute
Test View->Test View: Enter command
Test View->Test View: Step through actions

Game View->Results Page: Match Completes

Game View->Main Menu: Return to main menu
Test View->Main Menu: Return to Menu
Results Page->Main Menu: Return to main menu
